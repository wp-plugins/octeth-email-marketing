With this plug-in, blog owner will be able to "link" his Oempro account with WordPress and start accepting subscriptions.

== Description ==

Email marketing software with the features you need.
Provide the best possible email marketing experience to yourself and your clients with Oempro, The Email Marketing Software.

== Installation ==

Upload plugin folder to your blog (wp-content/plugins/), activate it, then enter your [oempro credentials]
